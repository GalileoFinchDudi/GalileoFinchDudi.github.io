  • 类型

    类型 例子 描述

# 字面值

字面值有点类似于常量，赋值之后就无法修改了。

但是通过字面值可以预设变量值的范围，例如：

let b: "value1" | "value2"
b = "value1"; // 赋值成功
b = "value2"; // 赋值成功
b = "value3"; // 赋值失败！

let c: boolean | string;
c = true; // 赋值成功
c = "value" // 赋值成功

# unknown

unknown是一个类型安全的any

let s :string ;
let e :unknown;
e = 'value';

s = e; // 此时赋值会报错: Type 'unknown' is not assignable to type 'string'.

// 想要赋值成功可以确保e必须是string类型，可以用如下写法
if (typeof e === 'string') {
    s = e;
}
// 类型断言， 用来告诉解析器变量的实际类型
s = e as string;
s = <string> e;

# Object

使用{}来制定对象中可以包含哪些属性 在属性名后面加上?标识可选属性


let b: {name: string, age?: number};
1
2
3

可以使用[]来表示任意类型的数组


// [propName: string]: unknown 表示任意类型的属性
let oc : {name: string, [propName: string]: unknown}
oc = {name: 'lisi', age: 12}
