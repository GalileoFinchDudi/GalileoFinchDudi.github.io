TypeScript-面向对象

面向对象：通过操作对象的方式来实现功能

数据在对象中被称为属性，对应的操作行为被称为方法也就是对象的功能。

# 类

就是对象的模型，对象是类的实例；

直接定义的属性是实例属性，需要通过创建对象的实例来访问；


class Person {
    name: string;
    age: number;
    static type = 'PersonType';
    constructor() {
        this.name = 'Galileo';
        this.age = 33;
    }
}

const person = new Person();

console.log(person); // Person {name: 'Galileo', age: 33}
console.log(Person.type); // PersonType

# static

使用static开头的属性是静态属性，可以直接通过类来访问。

class Person {
    name: string;
    age: number;
    static type = 'PersonType';
    constructor() {
        this.name = 'Galileo';
        this.age = 33;
    }
}

const person = new Person();

console.log(person); // Person {name: 'Galileo', age: 33}
console.log(Person.type); // PersonType

# readonly

readonly 开头属性的值只能读取，不能修改。












 



class Person {
    name: string;
    readonly age: number;
    constructor() {
        this.name = 'Galileo';
        this.age = 33;
    }
}

const person = new Person();
person.age = 12; // Error: Cannot assign to 'age' because it is a read-only property.

# 定义方法

class Person {
    name: string;
    age: number;
    constructor() {
        this.name = 'Galileo';
        this.age = 33;
    }

    sayHello() {
        console.log(`Hello, my name is ${this.name}. I am ${this.age} years old.`);
    }
}

const person = new Person();
person.sayHello();

同样，使用static开头的方法就是类方法，可以不构建实例直接调用。

class Person {
    name: string;
    age: number;
    constructor() {
        this.name = 'Galileo';
        this.age = 33;
    }

    static sayHello() {
        console.log(`Hello, I am a static method.`);
    }
}
Person.sayHello();
