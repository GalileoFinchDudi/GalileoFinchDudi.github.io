### 初始化npm环境

``` bash
npm init -y
```

### 安装相关依赖

``` bash
pnpm i -D webpack webpack-cli typescipt ts-loader
```

### 编写webpack配置文件

``` js
const path = require("path");

module.exports = {
  entry: "./src/index.ts",
  mode: "development",
  output: {
    path: path.resolve(__dirname, "dist"),
    filename: "index.js",
  },
  module: {
    rules: [
      {
        test: /\.ts$/,
        use: "ts-loader",
        exclue: /node_modules/,
      },
    ],
  },
};

```


### 编写typescript配置文件

详细的配置文档请参见[TypeScript-编译环境](./02.TypeScript-编译环境.md#compileroptions-编译选项)

``` json
{
    "compilerOptions": {
        "module": "ES6",
        "target": "ES6",
        "strict": true
    }
}
```
