## TypeScript 基础类型

- 类型

  |  类型   |       例子        |              描述              |
  | :-----: | :---------------: | :----------------------------: |
  | number  |     1,-33,2.5     |            任意数字            |
  | string  |     'hi',"hi"     |           任意字符串           |
  | boolean |    true,false     |      布尔值true或者false       |
  | 字面值🤔 |      其本身       |  限制变量的值就是该字面量的值  |
  |   any   |         *         |            任意类型            |
  | unknown |         *         |         类型安全的any          |
  |  void   | 空值（undefined） |     没有值（或undefined）      |
  | nerver  |      没有值       |          不能是任何值          |
  | object  | {name: 'Galileo'} |          任意的JS对象          |
  |  array  |      [1,2,3]      |           任意JS数组           |
  |  tuple  |       [4,5]       | 元素，TS新增类型，固定长度数组 |
  |  enum   |     enum{A,B}     |        美剧，TS新增类型        |

  
### 字面值

`字面值`有点类似于常量，赋值之后就无法修改了。

但是通过字面值可以预设变量值的范围，例如：

``` js
let b: "value1" | "value2"
b = "value1"; // 赋值成功
b = "value2"; // 赋值成功
b = "value3"; // 赋值失败！
```

或者通过字面值定义变量的类型，例如：

``` js
let c: boolean | string;
c = true; // 赋值成功
c = "value" // 赋值成功
```

### unknown

unknown是一个类型安全的any

``` js
let s :string ;
let e :unknown;
e = 'value';

s = e; // 此时赋值会报错: Type 'unknown' is not assignable to type 'string'.

// 想要赋值成功可以确保e必须是string类型，可以用如下写法
if (typeof e === 'string') {
    s = e;
}
// 类型断言， 用来告诉解析器变量的实际类型
s = e as string;
s = <string> e;
```

### Object

使用`{}`来制定对象中可以包含哪些属性
在属性名后面加上`?`标识可选属性

``` js

let b: {name: string, age?: number};

```

可以使用`[]`来表示任意类型的数组

``` js

// [propName: string]: unknown 表示任意类型的属性
let oc : {name: string, [propName: string]: unknown}
oc = {name: 'lisi', age: 12}

```

限制function的结构类型声明
语法：(形参: 类型, 形参: 类型) => 返回值

``` js
let fd: (a: number, b: number) => number;

fd = function (n1, n2) {
    return n1 + n2;
}
```

### Array

声明数组直接在类型后面加上`[]`

``` js

let arr: string[];

```

或者使用Array<类型>

``` js
let arr: Array<string>;
```

### Tuple

元组，固定长度的数组

``` js
let t: [string, number];
t = ['hello', 10];
```

### Enum

``` js
enum Gender {
    Male = 0,
    Female = 1
}

let ei : {name: string, gender: Gender}
ei = {
    name: 'zhangsan',
    gender: Gender.Male
}
console.log(ei.gender === Gender.Male)
```
